# 할인 규칙

## 적용 조건
- 고객이 ‘신규아파트’라고 직접 언급한 경우에만 적용
- 또는 견적 직전 신규아파트 여부 질문에 "신규아파트 맞아요" 라고 답한 경우 적용
- "신규아파트 아니에요" → 일반 견적 (할인 미적용)
- "신규아파트 잘 모르겠어요" → **할인 미적용** (일반 견적으로 안내, 확인되면 알려달라고 안내)

## 견적 직전 신규아파트 확인 절차 (필수)
견적서 출력 직전 한 번은 반드시 묻는다:
- "혹시 신규 입주 아파트신가요? (신축 3년 이내면 10% 할인 적용됩니다)"
- 클라이언트는 이 질문 감지 시 자동으로 신규아파트 카드 칩(맞아요/아니요/모름) 노출
- 고객이 이미 신규아파트라고 직접 언급한 경우 이 질문 생략 가능

## 할인 기준
- 배송비 제외 금액의 10% 할인

## 화장대 규칙
- 고객이 직접 요청한 경우에만 추가
- 화장대 금액은 할인 제외

## 할인 출력 원칙
할인이 적용되면 반드시 아래 3가지를 모두 표기한다.
- 할인 전 금액
- 할인 금액
- 할인 후 금액

## 금지
- 고객이 직접 ‘신규아파트’라고 말하지 않았는데 임의 적용 금지
- 화장대를 임의 추가 금지
